# Manage Device

## Purpose
Central hub for device configuration and maintenance.

## User Actions
- **Manage Firmware** - View version, update firmware
- **Manage Security Features** - PIN, duress, self-test
- **Enable/Disable Interfaces** - Toggle QR, USB, SD, SmartCard
- **Manage Display** - Brightness, theme settings
- **Manage Sounds** - Audio feedback settings
- **Wipe Device** - Factory reset (danger zone)

## Navigation
- Parent: main
- Contains most device administration functions
