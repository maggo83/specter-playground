# Enable/Disable Interfaces

## Purpose
Control which hardware interfaces are active. Security feature to limit attack surface.

## User Actions
Toggle each interface on/off:
- **QR Scanner** - Camera for QR code scanning
- **USB** - USB communication with host
- **SD Card** - SD card slot access
- **SmartCard** - SmartCard reader

## Security Context
Disabling unused interfaces reduces attack surface. Air-gapped setups typically disable USB.
